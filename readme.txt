Запускать приложение на dev-окружении с помощью команды: npm run dev;

Для сборки продакшн-версии используйте команду: npm run build;

Запустить JSON-server с помощью команды: npm run json-server;

Автоматически пересобирать проект при изменении исходных файлов в режиме разработки, запустить webpack-dev-server: npm run start.
